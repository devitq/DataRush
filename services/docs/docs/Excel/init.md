# Начало!

Microsoft Excel – это мощная программа для работы с электронными таблицами, которая является стандартом в деловом мире и не только.  Excel позволяет организовывать, анализировать и визуализировать данные, автоматизировать рутинные задачи и создавать отчеты. Этот краткий обзор поможет вам быстро освоить ключевые возможности Excel.
